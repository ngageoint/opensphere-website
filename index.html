About

OpenSphere is a an open source, pluggable GIS application built for handling and animating large volumes of data from a variety of different sources, servers, file formats, and more! It has both Desktop and Web variants that strive to bring a streamlined user experience to GIS analysis while still providing powerful functionality underneath the hood. It does all the heavy architectural lifting for getting an advanced mapping application off the ground, leaving your team free to to pursue connecting it to your back-end services and data formats.

Features

Desktop and Web

  • 3D Globe and 2D map views
  • 3D terrain
  • Large data volumes
  • Local file import (with drag and drop!)
  • Tiled layer display (WMS, XYZ, and Arc REST)
  • Vector data import (WFS, Arc REST, CSV, GeoJSON, GML, KML, KMZ, and Shapefiles)
  • Vector data export (CSV, KML, KMZ, and Shapefile)
  • Animation of both tile and vector sources that support time
  • Google Earth-esque KML tree display

Desktop Only

  • Higher data volumes
  • WAMI
  • Geopackage
  • Hi-res imagery import (GeoTIFF, NITF)

Web Only

  • Arbitrary projection support (ships with EPSG:4326 and EPSG:3857; 2D view only)
  • Vector data import for GPX
  • Heatmap generation from vector sources
